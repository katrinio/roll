Мой небольшой проект для учета пленочной фотографии.

### Зачем

Я снимаю на пленку не очень часто, поэтому легко забыть:

* какие пленки еще лежат неотснятыми;
* что сейчас заряжено в камеру;
* какие пленки уже отсняты;
* что уже проявлено;
* где лежат сканы;
* какой пленкой, камерой и лабораторией это было снято.

Хочется иметь одну простую систему, которая не требует помнить все в голове.

### Принципы

* Источник правды — обычная таблица.
* Архив фотографий хранится в папках.
* Код не заменяет таблицу, а только помогает с рутинными действиями.
* Система должна оставаться простой и понятной даже через несколько лет.

## Что хотелось бы автоматизировать
### Со временем проект может помочь:

* создать папку для новой проявки;
* сгенерировать шаблон заметок;
* открыть папку по номеру пленки;
* показать текущий статус всех пленок;
* собрать небольшую статистику.

Но все это должно оставаться необязательными помощниками, а не превращаться в отдельное приложение.

### Статусы пленки

* stock — лежит неотснятая;
* loaded — заряжена в камеру;
* shot — отснята;
* sent_to_lab — передана в лабораторию;
* developed — получены сканы;
* archived — разложена по архиву.

### Структура архива
```
~/Pictures/plenka
├── 2025
└── 2026
    ├── 04.18
    ├── e01638
    └── e02775
```
### Идея

Хочется получить спокойную каталог пленочных съемок, которая не отвлекает от самой фотографии.
