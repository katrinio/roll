# MVP

`roll` должен помогать быстро находить пленочные съемки по памяти.

Главная задача:

> найти папку со сканами по нескольким словам: человек, место, событие, настроение.

## Основной сценарий

Я получаю сканы из фотостудии, кладу их в архив и запускаю:

```bash
rl init ~/Pictures/plenka
rl index ~/Pictures/plenka/2026/e02775
rl search kir balcony
```

## Что входит в MVP

- глобальная конфигурация знает только список архивов;
- архив хранит `.roll/` и словари у себя;
- `index` запрашивает недостающее из словарей;
- `search` ищет по фильмам, камерам, признакам и ключевым словам;
- `doctor` показывает проблемы;
- `normalize` приводит имена папок к единому виду.

## Что не входит в MVP

- синхронизация между машинами;
- облако;
- веб-интерфейс;
- сложная миграция старых форматов;
- автоматическая обработка изображений.

## Правило

Если новая функция не помогает быстрее найти конкретную пленку по памяти, она не входит в MVP.
